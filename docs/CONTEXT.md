Vendura is a developer-first e-commerce framework for Next.js.

It provides deterministic cart, checkout, order state, payment verification,
and inventory correctness.

Vendura core owns:
- Checkout lifecycle
- Order state transitions
- Payment verification
- Rate limiting
- Security invariants

Vendura does not provide:
- UI components
- Auth systems
- CMS
- Shipping
- Themes

Vendura is infrastructure, not an app.
